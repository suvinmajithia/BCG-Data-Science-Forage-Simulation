  • What it does: Calculates the mean, max, and standard deviation for the historical price components.

  • The Goal: To determine if energy prices have been stable or volatile over time, which is key context for analyzing customer price sensitivity.

This block of code performs the Numeric Data Visualization.

It creates four distribution charts (histograms) to clearly show the range and most common values for the main numerical features, especially consumption and revenue.

  • Simple Overview of Code Actions

  • Sets up the Charts: The code selects four key numerical columns (cons_12m, net_margin, etc.) that are expected to be heavily skewed (unbalanced).

  • Applies the Log Scale: The most important action is using the logarithmic transformation (np.log1p).

  • Why? Because consumption and margin values have extreme outliers (e.g., one client consuming 1,000,000 units and 90% of clients consuming less than 5,000 units).

  • If plotted normally, the chart would be useless (a huge spike at the left, and a flat line everywhere else). The log scale squashes the large values so one can actually see the shape of the distribution for the average customer.

  • Histograms: It then plots the new, visible distribution for each column.

  • The Goal: To visually confirm the high skewness and show that most customers are clustered at the low end of consumption and margin.

In[]:
# Numeric Features (Log Distribution for Skewness)
# Use a log transformation to visually inspect the distributions of the skewed consumption and financial data.
print("\nNumeric Features: Log Distribution (Checking for Skewness/Outliers)")
# Key numerical columns prone to high skewness
numeric_cols_to_plot = ['cons_12m', 'forecast_cons_12m', 'net_margin', 'num_years_antig']

plt.figure(figsize=(16, 8))

for i, col in enumerate(numeric_cols_to_plot):
    plt.subplot(2, 2, i + 1)
    # Apply np.log1p(x) (log(1+x)) for robust visualization as it handles zeros
    sns.histplot(df_client[col].apply(lambda x: np.log1p(x)).dropna(), kde=True, bins=50)
    plt.title(f'Log Distribution of {col}')
    plt.xlabel(f'log(1 + {col})')
    plt.ylabel('Frequency')

plt.tight_layout()
plt.show()
print("-" * 70)
